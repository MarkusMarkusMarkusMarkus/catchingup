# Catching Up

Field notes on GenAI. Live at [catchingup.now](https://catchingup.now)

## Publishing a new post

1. Write the post in Google Docs
2. Copy the content into `posts/template.html`
3. Save as `posts/32-post-title.html` (next number in sequence)
4. Add keywords to the `<meta name="keywords">` tag in the file
5. Run the build script:
   ```bash
   node build.js
   ```
6. Commit and push:
   ```bash
   git add .
   git commit -m "post #32: post title"
   git push
   ```
7. Live in ~60 seconds.

## File naming

Posts follow the pattern: `{number}-{slug}.html`

Examples:
- `01-getting-it.html`
- `32-new-post-title.html`

The number determines the order in the sidebar. Always zero-pad to two digits.

## Structure

```
/
  index.html       ← homepage (don't touch)
  posts.json       ← auto-generated by build.js (don't touch)
  build.js         ← run this before pushing
  posts/
    template.html  ← copy this for each new post
    01-getting-it.html
    02-innovative-laggard.html
    ...
  assets/
    images/        ← drop images here, reference as ../assets/images/filename.png
```

## GitHub Pages setup

1. Go to repo Settings → Pages
2. Source: Deploy from branch → `main` → `/ (root)`
3. Custom domain: `catchingup.now`
4. In Porkbun DNS, add a CNAME record pointing to `{your-github-username}.github.io`
